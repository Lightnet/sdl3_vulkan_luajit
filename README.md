# SDL3 Vulkan LuaJIT

# Required:
 * VS2022 (It need NMake so need to install c/c++ that has it.)
 * CMake
 * vulkan-sdk-1.4.304.1 (current build)

# Information:
 Prototyping build for wrapper SDL3, Vulkan for Lua script.

```
sdl3_vulkan_luajit/

├── include/

├── src/
│   ├── main.c
│   └── sdl3_luajit.c

```
